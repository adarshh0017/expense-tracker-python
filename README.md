# Expense Tracker (Python + SQLite)

## Overview
A simple command-line expense tracker built using Python and SQLite.  
It allows users to add, view, and calculate total expenses.

## Features
- Add expenses with date, category, amount, and description
- View all expenses
- Calculate total spending
- Menu-driven CLI application

## Technologies Used
- Python
- SQLite

## How to Run
1. Open terminal
2. Run:
   python expense_tracker.py

## Example Output
1. Add Expense  
2. View Expenses  
3. Total Expense  
4. Exit
